## Overview

This project is a full-stack AI-powered code sharing platform where developers can:

- Register and log in securely
- Share code snippets and programming questions
- Receive AI-generated code review comments using Gemini API
- Like and comment on posts
- View user profiles and personal posts

The goal of the project is to create a developer-focused social platform that combines community interaction with AI-assisted code feedback.

---

## Tech Stack

### Frontend
- React.js
- Vite
- React Router DOM
- Context API
- CSS (global styles + component styles)

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Google Gemini API (Gemini 2.5 version)

---

## Features

### Authentication
- User registration
- User login
- JWT-based authentication
- Protected routes

### Posts System
- Create new code posts
- View home feed of posts
- Like posts
- Delete posts
- View post details

### Comments System
- Add comments to posts
- Like comments
- Delete comments

### AI Review System
- Gemini API analyzes posted code
- AI generates smart code review feedback
- Suggestions for bugs, improvements, and best practices

### Profile System
- View personal profile
- View user-created posts
